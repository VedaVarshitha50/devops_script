## GIT Commands ##

1. Clone a Project
    git clone <project_url>

2. Check the status of change
    git status 

3. Add change files to the git 
    git add .

4. Add commit message
    git commit -m "message"

5. Push the change 
    git push -u origin <branch_name>

6. Create a Branch and checkout to branch
    git checkout -b <branch_name>

7. To check in which branch we are currently
    git branch
 
8. To switch between branches
    git checkout <branch_name_to be created>
